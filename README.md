## 💻 Sobre o Curso
O Python é uma das linguagens mais utilizadas no mundo da tecnologia e um dos melhores pontos de partida para quem está começando, por sua simplicidade e ampla aplicação em áreas como dados, automação e inteligência artificial.

O trilha é introdutória, prática e acessível, com foco nos fundamentos da lógica de programação e da linguagem Python. Para quem já possui experiência em outras linguagens, o curso atua como uma excelente ferramenta de atualização, permitindo relembrar algoritmos consolidados e aplicar rapidamente a nova sintaxe em soluções modernas.

## 🚀 Como Executar
Para rodar os desafios localmente, siga os passos abaixo:

1- Pré-requisitos: <br>
* Certifique-se de ter o Python 3.x instalado em sua máquina.<br>
* Recomendado o uso do VS Code.<br>

2- Clone o repositório: 

```
git clone https://github.com/seu-usuario/Introducao-Python-WoMakersCode.git
```

3- Navegue até a pasta do desafio desejado: 

```
cd "Nome da Pasta do Desafio"
```

4- Execute o script:

```
"python nome_do_arquivo.py
```
